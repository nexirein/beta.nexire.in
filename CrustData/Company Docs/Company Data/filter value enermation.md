## Last Funding Round Type
For a complete list of last funding round types, refer to:
[Last Funding Round Type JSON](/examples/company-discovery/last_funding_round_type.json)

Common values:
- `"seed"`, `"pre_seed"`
- `"angel"`
- `"series_a"`, `"series_b"`, `"series_c"`, `"series_d"`, `"series_e"`, `"series_f"`, `"series_g"`, `"series_h"`, `"series_i"`, `"series_j"`
- `"series_unknown"`
- `"private_equity"`
- `"corporate_round"`
- `"convertible_note"`
- `"debt_financing"`
- `"equity_crowdfunding"`, `"product_crowdfunding"`
- `"grant"`
- `"initial_coin_offering"`
- `"post_ipo_equity"`, `"post_ipo_debt"`, `"post_ipo_secondary"`
- `"secondary_market"`
- `"undisclosed"`
- `"non_equity_assistance"`
