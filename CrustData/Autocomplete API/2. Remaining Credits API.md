# Remaining Credits API

### [ 🚀 Try Now ](/api#tag/auxiliary-api/GET/user/credits)

This endpoint allows you to check your remaining API credits balance. You can use this to monitor your credit usage and ensure you have sufficient credits for your API requests.

You can also view your credits and usage details through the <a href="app.crustdata.com/screenerv2/developers">Dashboard</a>

## Endpoint
```
GET /user/credits
```

## Request Parameters

This endpoint does not require any parameters.

## Credit Usage

- This endpoint does not consume any credits

## Example Request

<details id="1-check-credits-balance">
<summary>1. Check Credits Balance</summary>

```bash
curl 'https://api.crustdata.com/user/credits' \
--header 'Authorization: Token $authToken'
```
</details>

## Example Responses

<details id="1-success-response">
<summary>Success Response</summary>

### Successful Response (200 OK)
```json
{
  "credits": 9406
}
```
</details>

## Response Fields

| Field    | Type    | Description                                    |
|----------|---------|------------------------------------------------|
| `credits`| integer | The number of API credits remaining in your account |
