# Logos Directory

Upload your logo image to this folder and name it `logo.png`.
It will be standardly picked up by the sidebar and login components.

If you don't upload a logo, the generic "Zap" icon from lucide-react will be shown as a fallback.

Make sure you restart the dev server or do a hard refresh in your browser if you've added the file while Next.js is running, though typically Next.js `public` directory changes are picked up relatively quickly.
