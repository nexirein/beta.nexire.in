## The One UX Rule That Ties It All Together

Every purple element = "this costs something or does something important." Every zinc/gray element = "this is information." Every green element = "this is a positive signal." Keep this system consistent across the entire app — Indian recruiters trust software that feels organized and intentional, not random. The moment a user understands your visual grammar intuitively (green = good match, purple = action), you've crossed from "nice design" to "professional tool."
